# Assignment 4: Data Collection and Analysis

## 1. 데이터 수집 및 분석의 목표
- 비트세이버 곡들에서 이번 데이터 분석의 기준이 되는 mapping 데이터를 추출하여 시각화 후 분석.
- 고유 번호, (곡 제목), 중요 메타데이터(BPM 등), 난이도(1~5단계), mapping 데이터 등으로 데이터를 압축, 하나로 모아 하나의 json 형식 파일로 추출. 

## 2. RAW 데이터 정의
출처: https://beatsaver.com/ (비트세이버의 커스텀 레벨 공유 사이트)
구조(예시): 
* 📁 1eaf3 (BOY - King Gnu) <- 
    * 📄 info.dat
    * 📄 ExpertStandard.dat
    * 📄 ExpertPlusStandard.dat
      (관련성이 없는 일부 파일 생략됨)
- 하나의 곡은 하나의 폴더에 정리되며 각 난이도는 **json문법**으로 작성된 **.dat파일**에 저장. 또한 폴더는 사이트에서 부여한 맵의 코드와 괄호 안의 이름으로 표시.
- info.dat: 곡의 메타 정보가 담겨있는 파일. 곡 자체에 대한 정보(제목, 작곡가 등), BPM, Audio offset, 각 난이도에 대한 정보, **각 난이도의 Mapping 파일 경로** 등이 포함.
- ExpertStandard.dat: Expert 난이도의 Mapping 파일. 각 난이도의 Mapping 파일은 일반적으로 **{난이도}Standard.dat**의 이름을 가짐.
## 3. 문제 발생
- 시간 부족으로 인하여 실제 데이터가 아닌 실제 데이터의 메타 데이터를 추출하는데 그침
- 또한 기본적인 통계 분석은 나왔지만 시각화 할 자료가 부족함.
## 4. 분석
- 메타 데이터 상으로 포집된 데이터인 375개의 곡의 평균 BPM은 155, 최솟값은 80 최댓값은 300
- 앞으로 이 메타 데이터를 바탕으로 각 레벨에 대한 정보를 추출해야 함.
